# Market Regime Analysis

## Overview

This project explores the identification and characterization of market regimes in XAUUSD using exploratory data analysis, clustering techniques, Markov chains, and risk assessment methodologies.

The objective is to understand how market states emerge, persist, and transition over time, providing insights for decision-making under uncertainty.

---

## Objectives

- Identify distinct market regimes
- Characterize regime behavior
- Analyze state transitions using Markov chains
- Evaluate persistence and risk across regimes
- Build a reproducible analytics framework

---

## Methodology

1. Data Collection and Preparation
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Clustering Analysis
5. Markov Transition Analysis
6. Risk Assessment (in progress)
