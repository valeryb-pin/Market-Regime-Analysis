# Feature Engineering

Modules responsible for generating market features used by the clustering model.

## Files

- build_features.py
- efficiency.py
- imbalance.py
- liquidity.py
- microstructure.py
- movement.py
- orderflow.py

## Categories

The generated features represent:

- Market Efficiency
- Liquidity
- Volatility
- Market Activity
- Order Flow
- Microstructure
- Persistence

These features form the input space used for regime identification.
